new PropHazards()

Proportional Hazards Model with a constant hazard function. Uses Newtons method to compute the weights. Before use: include BLAS library.

Example 

// import analytics module
var analytics = require('qminer').analytics;
// create a Proportional Hazard model
var hazard = new analytics.PropHazards();
