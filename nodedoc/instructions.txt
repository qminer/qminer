Look at ht module for example.
1. each submodule should include all C++ code documented in a single header file using JSDOC located in src/nodejs/module_name/module_source.h
2. optionally additional method implementations should be src/nodejs/scripts/module_source.js and the functions should be documented using JSDOC. All the code that is to be copied into the aggregate js file should be between //!STARTJSDOC and //!ENDJSDOC

Use makedoc.js to aggregate the code for a single module, here's an example:
makedoc usage:
   node makedoc header_file js_file aggregate_file
Example:
```
node makedoc.js src/nodejs/ht/ht_nodejs.h src/nodejs/scripts/ht.js nodedoc/htdoc.js
```

Install jsdoc:
```
npm install jsdoc -g
```

jsdoc nodedoc/htdoc.js -d nodedoc/htdoc
