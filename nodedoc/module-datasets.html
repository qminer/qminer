datasets

Source: datasetsdoc.js:2

Datasets module includes some standard toy datasets. In addition, this module also includes various random sample generators that can be used to build artificial datasets of controlled size and complexity.

Examples

Asynchronous loading

var qm = require('qminer');
var datasets = qm.datasets;

// Create clean base
var base = new qm.Base({ mode: 'createClean' });

// Load Iris dataset in async way
datasets.loadIris(base, function (err, store) {
    if (err) throw err;
    // Sucessfully loaded database
    base.close();
});

Synchronous loading

var qm = require('qminer');
var datasets = qm.datasets;

// Create clean base
var base = new qm.Base({ mode: 'createClean' });

// Load Iris in sync way.
var Iris = datasets.loadIrisSync(base);
// Sucessfully loaded database
base.close();
